# 🚀 PixelBoost AI - Complete Image & Video Upscaler

**Upload → Upscale → Generate Metadata → Download**

## Features
- 🖼️ Upscale Images 2x-8x
- 🎬 Upscale Videos 2x-4x
- 🤖 AI Metadata Generator (Title, Keywords, Description, Category, Alt Text)
- 🌍 10 Languages Support
- 📐 Multi-Size Versions (1080p, 2K, 4K)
- 📊 CSV Export for Adobe Stock
- 🔑 API Key stored in Secrets (secure)

## Supported Formats
- Images: JPG, JPEG, PNG
- Videos: MP4, MOV, AVI

## How to Use
1. Upload images or videos
2. Choose scale (4x recommended)
3. Generate metadata (optional)
4. Process & download ZIP
